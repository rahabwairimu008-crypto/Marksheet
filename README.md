# Marksheet Project

##  Overview
This project is a simple **Marksheet Project** that calculates and displays student grades based on input marks.  
It is designed for academic use, providing a clear and professional output similar to a real-world marksheet.

##  Features
- Input student details eg;name
- Automatic calculation of:
  - Total marks
  - Averages
  - Grade classification
- Cleanly formatted output resembling a professional marksheet
- Error handling for invalid inputs

##  Technologies Used
- Java (console-based program, no arrays/loops )
- GitHub for version control and documentation
